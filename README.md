# IPL-Analysis-Dashboard-PowerBI
IPL Data Analysis (2008–2025) is a Power BI dashboard that analyzes 17 IPL seasons through interactive visualizations and key performance metrics. It showcases team standings, champions, player statistics, batting and bowling records, venue insights, and season-wise trends using Power BI, DAX, and Power Query.
