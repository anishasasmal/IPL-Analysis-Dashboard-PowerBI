# IPL-Analysis-Dashboard-PowerBI
IPL Data Analysis (2008–2025) is a Power BI dashboard that analyzes 17 IPL seasons through interactive visualizations and key performance metrics. It showcases team standings, champions, player statistics, batting and bowling records, venue insights, and season-wise trends using Power BI, DAX, and Power Query.

## 📌 Project Overview
This dashboard provides season-wise IPL insights from 2008–2025, including team performance, player achievements, and tournament statistics through interactive visuals.

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel
- KPIs

## 📊 Dashboard Features
- Season-wise filter
- Champions & Runner-up
- Orange Cap & Purple Cap
- Points Table
- Total Matches
- Matches Lost
- Matches Won
- Total Teams
- Total Fours & Sixes
- Centuries & Half-centuries
- Venues Used
